## 3. 주요 유저 플로우

### 플로우 1: 루틴 체크하기 (가장 빈번)
```
홈 화면 → 루틴 체크박스 탭 → 완료 애니메이션 → 프로그레스 업데이트
```

### 플로우 2: 루틴 추가
```
홈 화면 → FAB(+) 탭 → 바텀시트 "루틴 추가" 선택
→ 폼 (제목 입력, 카테고리 선택) → 저장 → 홈 화면에 추가됨
```

### 플로우 3: 할 일 추가
```
홈 화면 → FAB(+) 탭 → 바텀시트 "할 일 추가" 선택
→ 폼 (제목 입력, 카테고리 선택) → 저장 → 홈 화면에 추가됨
```

### 플로우 4: 루틴 수정/삭제
```
홈 화면 → 루틴 항목 오른쪽 스와이프 → 수정 버튼 → 수정 폼
홈 화면 → 루틴 항목 왼쪽 스와이프 → 삭제 버튼 → 확인 다이얼로그 → 삭제
```

### 플로우 5: 통계 확인
```
바텀 탭 "통계" → 기간 선택 (일/주/월) → 차트 확인
→ 하단 카테고리별 분석 확인
```

### 플로우 6: 카테고리 관리
```
바텀 탭 "설정" → "카테고리 관리" → 카테고리 목록
→ 추가/수정/삭제
```

### 플로우 7: 자정 자동 초기화 (자동)
```
자정 도달 → 새 DailyRecord 생성 → 모든 루틴 미완료로 표시
→ 이전 기록 보존 (통계용)
```

### 플로우 8: 11시 스마트 알림 (자동)
```
22:55 → 백그라운드 체크 → 미완료 항목 존재?
  ├── Yes → "아직 N개의 할 일이 남았어요!" 알림 → 탭 시 홈 화면
  └── No → 알림 없음 (스마트 무음)
```

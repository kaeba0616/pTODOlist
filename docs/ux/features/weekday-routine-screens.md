# F10: 요일별 루틴 - Screens

## 루틴 추가/수정 바텀시트 (활성 요일 섹션 추가)

```
┌─────────────────────────────────┐
│  (기존: 제목, 카테고리, 아이콘,  │
│   우선순위, 비활성화)             │
│                                 │
│  활성 요일                       │  제목 (bold)
│  ┌───┬───┬───┬───┬───┬───┬───┐ │
│  │ 월 │ 화 │ 수 │ 목 │ 금 │ 토 │ 일 │ │  FilterChip 7개
│  └───┴───┴───┴───┴───┴───┴───┘ │  선택=primary, 미선택=grey
│                                 │
│  (기존: 세부 항목, 저장 버튼)     │
└─────────────────────────────────┘
```

### 요일 칩 UI 요소
- **타입**: FilterChip (7개 가로 배치)
- **레이블**: 월, 화, 수, 목, 금, 토, 일
- **선택 상태**:
  - ON: primary color 배경
  - OFF: grey[200] 배경
- **기본 상태**: 전체 선택 (= 매일)
- **전체 선택 단축**: 모두 해제 상태에서 탭 시 전체 선택으로 복원

## 홈 화면 - 루틴 타일 변경

### 요일 표시
- 매일이 아닌 루틴: 제목 옆에 작은 요일 텍스트 표시
  - 예: "아침 운동 (월수금)"
- 매일 루틴: 추가 표시 없음

```
│ 🏃○ 아침 운동 (월수금)  🔺 3  🔴 │
│ 📚○ 영어 공부              2  🔵 │  매일이면 표시 없음
```

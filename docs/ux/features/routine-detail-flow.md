# 루틴 상세 (우선순위 & 아이콘) - UX Flow

## 개요
루틴에 우선순위(높음/보통/낮음)와 아이콘을 설정하여
홈 화면에서 시각적으로 빠르게 구별할 수 있다.

---

## 플로우 1: 루틴 추가 시 우선순위/아이콘 선택
```
루틴 추가 바텀시트
→ 제목 입력
→ 카테고리 선택
→ 아이콘 선택 (선택, 기본=check_circle)
    → 아이콘 그리드에서 탭하여 선택
→ 우선순위 선택 (기본=보통)
    → 3단계 SegmentedButton: 낮음 / 보통 / 높음
→ 세부 항목 (기존)
→ 저장
```

## 플로우 2: 루틴 수정 시 변경
```
홈 화면 → 루틴 탭 → 수정 바텀시트
→ 기존 아이콘/우선순위가 미리 선택됨
→ 변경 후 저장
```

---

## 우선순위 규칙
- **높음** (priority: 2): 빨간 뱃지, 목록 상단 정렬
- **보통** (priority: 1): 기본 표시 (뱃지 없음)
- **낮음** (priority: 0): 회색 표시

## 홈 화면 정렬
- 같은 카테고리 내에서 우선순위 높은 순 → order 순

## 아이콘 규칙
- 미리 선별된 24개 Material Icons 제공
- 선택하지 않으면 기본 아이콘 (check_circle)
- 아이콘은 홈 화면 타일 leading에 표시 (체크 상태와 결합)

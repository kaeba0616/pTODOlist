# F3: 추가 할 일 - Screens

## 할 일 추가 바텀시트

```
┌─────────────────────────────────┐
│  ─── (드래그 핸들)               │
│                                 │
│  할 일 추가                      │
│                                 │
│  제목                           │
│  ┌─────────────────────────┐   │
│  │ 마트 장보기               │   │  TextField
│  └─────────────────────────┘   │
│                                 │
│  카테고리                        │
│  ┌─────────────────────────┐   │
│  │ 🟢 생활              ▼  │   │  DropdownButton
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │         저장              │   │  ElevatedButton
│  └─────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
```

## 홈 화면 내 할 일 타일 (TaskTile)

```
┌─────────────────────────────────┐
│  ○  마트 장보기          🟢     │  미완료
│  ◉  이메일 답장          🟡     │  완료 (취소선 + 투명도 60%)
└─────────────────────────────────┘
```

- 완료된 항목은 하단으로 정렬
- 자정에 초기화되지 않음

# 홈 위젯 (F14) - Screen Specs

## W1: 홈 위젯 (중형 4x2)

```
┌──────────────────────────────────────────┐
│  📋 pTODOlist    3월 19일 (목)    3/7    │  ← 헤더
├──────────────────────────────────────────┤
│  ██████████████████░░░░░░░░  43%         │  ← 진행률 바
├──────────────────────────────────────────┤
│  ✅  아침 운동                           │  ← 완료 루틴
│  ✅  영어 공부                           │  ← 완료 루틴
│  ✅  물 2L 마시기                        │  ← 완료 루틴
│  ☐   코딩 연습                           │  ← 미완료 루틴
│  ☐   독서 30분                           │  ← 미완료 루틴
│                              +2개 더     │  ← 추가 항목 안내
└──────────────────────────────────────────┘
```

### UI 요소 상세

**헤더 영역**
- 좌측: 앱 아이콘 (작은 크기) + "pTODOlist" 텍스트
- 중앙: 오늘 날짜 (예: "3월 19일 (목)")
- 우측: 달성률 (예: "3/7")
- 배경: surface 색상
- 텍스트 크기: 12sp
- 전체 영역 탭 → 앱 열기

**진행률 바**
- 높이: 6dp
- 모서리: 3dp radius
- 트랙: surfaceContainerHighest
- 바: 달성률에 따른 색상
  - 0~49%: `#F59E0B` (warning/amber)
  - 50~99%: `#4F46E5` (primary/indigo)
  - 100%: `#10B981` (success/green)
- 우측: 퍼센트 텍스트 (12sp)

**루틴 리스트**
- 최대 5개 항목 표시
- 각 항목: 체크박스 (18dp) + 제목 텍스트 (14sp)
- 완료 항목:
  - 체크박스: 채워진 원 + 체크마크 (`#10B981`)
  - 텍스트: 취소선 + 50% opacity
- 미완료 항목:
  - 체크박스: 빈 원 테두리 (`#4F46E5`)
  - 텍스트: 일반
- 정렬: 미완료 먼저, 완료 아래
- 체크박스 탭 → 루틴 토글

**추가 항목 안내**
- 루틴이 5개 초과 시 표시
- 텍스트: "+N개 더" (12sp, onSurfaceVariant)
- 우측 정렬

## 위젯 스타일링

### 색상 (Android RemoteViews 호환)

| 요소 | 라이트 모드 | 다크 모드 |
|---|---|---|
| 위젯 배경 | `#FFFFFF` | `#1F2937` |
| 헤더 배경 | `#F9FAFB` | `#111827` |
| 헤더 텍스트 | `#374151` | `#D1D5DB` |
| 리스트 텍스트 | `#111827` | `#F9FAFB` |
| 구분선 | `#E5E7EB` | `#374151` |
| 체크박스 완료 | `#10B981` | `#10B981` |
| 체크박스 미완료 | `#4F46E5` | `#818CF8` |

### 레이아웃

- 위젯 크기: 4x2 그리드 셀 (약 250x110dp)
- 모서리: 16dp radius (Android 12+ 자동 적용)
- 내부 패딩: 12dp
- 항목 간 간격: 4dp
- 체크박스-텍스트 간격: 8dp

## Android 네이티브 구현 참고

### RemoteViews 제약사항
- RemoteViews는 제한된 위젯만 지원:
  - `LinearLayout`, `RelativeLayout`, `FrameLayout`
  - `TextView`, `ImageView`, `ProgressBar`
  - `ListView` (API 11+)
- Custom View 불가 → 표준 Android 위젯으로 구현
- 체크박스는 `ImageView` + `PendingIntent`로 구현 (실제 Checkbox 불가)

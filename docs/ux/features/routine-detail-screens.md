## 홈 화면 - 루틴 타일 변경

### 기존
```
│ ○ 아침 운동              2  🔴 │
```

### 변경 후
```
│ 🏃○ 아침 운동         🔺 2  🔴 │  아이콘 + 체크 + 제목 + 우선순위뱃지 + 서브태스크수 + 카테고리
│ 🏃✓ 아침 운동         🔺 2  🔴 │  완료 시
```

### UI 요소 상세

**아이콘 (leading 확장)**:
- 작은 아이콘 (18px) + 체크 아이콘 (24px) 가로 배치
- 완료 시: 아이콘 grey, 체크 green

**우선순위 뱃지 (trailing)**:
- 높음: 🔺 빨간색 (Icons.arrow_drop_up, Colors.red)
- 보통: 표시 없음
- 낮음: 🔻 회색 (Icons.arrow_drop_down, Colors.grey)

# F1: 카테고리 관리 - Screens

## 카테고리 목록 화면 (설정 > 카테고리 관리)

```
┌─────────────────────────────────┐
│  ←  카테고리 관리                 │  AppBar
├─────────────────────────────────┤
│                                 │
│  🔴 운동                    3개  │  CategoryTile
│  ─────────────────────────────  │
│  🔵 공부                    2개  │
│  ─────────────────────────────  │
│  🟡 업무                    1개  │
│  ─────────────────────────────  │
│  🟢 생활                    4개  │
│  ─────────────────────────────  │
│  🟣 기타                    0개  │
│  ─────────────────────────────  │
│                                 │
│  ┌─────────────────────────┐   │
│  │   + 카테고리 추가         │   │  TextButton.icon
│  └─────────────────────────┘   │
│                                 │
├─────────────────────────────────┤
│   🏠 오늘    📊 통계    ⚙ 설정   │
└─────────────────────────────────┘
```

### CategoryTile UI 요소
- 왼쪽: 색상 원형 도트 (12dp)
- 중앙: 카테고리 이름 (bodyLarge)
- 우측: 연결된 루틴/할일 수 (labelSmall, gray)
- 탭: 수정 바텀시트 열기
- 왼쪽 스와이프: 삭제 버튼 노출

## 카테고리 추가/수정 바텀시트

```
┌─────────────────────────────────┐
│  ─── (드래그 핸들)               │
│                                 │
│  카테고리 추가                    │  제목 (수정 시 "카테고리 수정")
│                                 │
│  이름                           │
│  ┌─────────────────────────┐   │
│  │ 운동                     │   │  TextField
│  └─────────────────────────┘   │
│                                 │
│  색상                           │
│  🔴 🔵 🟡 🟢 🟣 ⚫ 🟤 🩷      │  ColorPicker (프리셋)
│                                 │
│  ┌─────────────────────────┐   │
│  │         저장              │   │  ElevatedButton (Primary)
│  └─────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
```

### UI 요소
- **이름 입력**: TextField, maxLength 20, 빈 값 시 저장 버튼 비활성
- **색상 선택**: 8개 프리셋 색상의 원형 버튼 그리드
  - 프리셋: Red, Blue, Amber, Green, Purple, Gray, Brown, Pink
  - 선택된 색상: 체크 아이콘 오버레이
- **저장 버튼**: 이름이 비어있으면 비활성

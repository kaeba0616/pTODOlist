# F8: 설정 & 데이터 관리 - UX Flow

## 진입 경로
바텀 탭 "설정"

---

## 설정 항목

### 1. 카테고리 관리
```
설정 → "카테고리 관리" → CategoryListView (F1에서 구현 완료)
```

### 2. 알림 설정
```
설정 → 11시 알림 ON/OFF 토글 (F7에서 구현 완료)
```

### 3. 데이터 보관 기간
```
설정 → "데이터 보관 기간" → 드롭다운 선택
  - 3개월
  - 6개월 (기본)
  - 1년
  - 무제한
→ 앱 실행 시 보관기간 초과 DailyRecord 자동 삭제
```

### 4. 테마
```
설정 → "테마" → 드롭다운 선택
  - 시스템 설정 (기본)
  - 라이트
  - 다크
```

### 5. 앱 정보
```
설정 → "앱 정보" → 버전 표시
```

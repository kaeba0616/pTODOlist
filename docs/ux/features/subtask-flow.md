# F9: 세부 항목(Subtask) & 편집 - UX Flow

## 개요
루틴과 할 일에 세부 항목(subtasks)을 추가할 수 있고,
홈 화면에서 항목을 탭하여 수정할 수 있다.

---

## 진입 경로
- 홈 화면 FAB(+) → 루틴/할 일 추가 시 세부 항목 입력
- 홈 화면 항목 탭 → 수정 바텀시트에서 세부 항목 편집
- 루틴/할 일 수정 바텀시트 내 세부 항목 섹션

---

## 플로우 1: 세부 항목과 함께 루틴 추가
```
홈 화면 → FAB(+) → "루틴 추가"
→ 루틴 추가 바텀시트
  → 제목 입력
  → 카테고리 선택
  → 세부 항목 추가 (선택)
    → 텍스트 입력 → "+" 버튼 또는 Enter
    → 항목이 목록에 추가됨
    → "X" 버튼으로 개별 삭제
  → "저장" 버튼
→ 홈 화면에 새 루틴 + 세부 항목 표시
```

## 플로우 2: 세부 항목과 함께 할 일 추가
```
홈 화면 → FAB(+) → "할 일 추가"
→ 할 일 추가 바텀시트
  → 제목 입력
  → 카테고리 선택
  → 세부 항목 추가 (선택)
  → "저장" 버튼
→ 홈 화면에 새 할 일 + 세부 항목 표시
```

## 플로우 3: 홈에서 루틴 수정
```
홈 화면 → 루틴 타일 탭 (체크 아이콘 외 영역)
→ 루틴 수정 바텀시트 (기존 값 채워짐)
  → 제목 수정
  → 카테고리 변경
  → 비활성화 토글
  → 세부 항목 추가/삭제
  → "저장" 버튼
→ 홈 화면 업데이트
  → 비활성화 시 목록에서 제거 + DailyRecord에서도 제거
```

## 플로우 4: 홈에서 할 일 수정
```
홈 화면 → 할 일 타일 탭 (체크 아이콘 외 영역)
→ 할 일 수정 바텀시트 (기존 값 채워짐)
  → 제목 수정
  → 카테고리 변경
  → 세부 항목 추가/삭제
  → "저장" 버튼
→ 홈 화면 업데이트
```

---

## 인터랙션 규칙

### 체크 vs 편집 구분
- **체크 아이콘 탭**: 완료/미완료 토글 (기존 동작)
- **타일 나머지 영역 탭**: 수정 바텀시트 열기

### 세부 항목 제한
- 각 세부 항목 최대 30자
- 세부 항목 수 제한 없음 (UI 스크롤 처리)
- 빈 텍스트는 추가 불가

### 홈 화면 표시
- 세부 항목이 있는 경우 항목 수 뱃지 표시
- 항목 하단에 세부 항목 인라인 표시 (↳ 아이콘 + 텍스트)
- 완료된 루틴/할 일의 세부 항목도 취소선 처리

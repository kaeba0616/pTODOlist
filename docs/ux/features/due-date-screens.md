## 홈 화면 - 할 일 섹션 변경

```
│  추가 할 일                      │
│                                  │
│ ○ 보고서 제출     D+2  🔴  🟡   │  overdue (빨간 뱃지)
│ ○ 회의 준비       D+1  🔴  🔵   │  overdue
│ ─────────────────────────────── │
│ ○ 마트 장보기           1  🟢   │  오늘 할 일
│ ✓ 이메일 확인              🟢   │  완료
```

### D+N 뱃지
- 색상: Colors.red
- 크기: 11px
- 위치: trailing, 서브태스크 수 왼쪽

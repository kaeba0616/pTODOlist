# F9: 세부 항목(Subtask) & 편집 - Screens

## 루틴/할 일 추가·수정 바텀시트 (세부 항목 섹션 추가)

```
┌─────────────────────────────────┐
│  ─── (드래그 핸들)               │
│                                 │
│  루틴 추가 / 루틴 수정           │
│                                 │
│  제목                           │
│  ┌─────────────────────────┐   │
│  │ 아침 운동                 │   │  TextField
│  └─────────────────────────┘   │
│                                 │
│  카테고리                        │
│  ┌─────────────────────────┐   │
│  │ 🔴 운동              ▼  │   │  DropdownButton
│  └─────────────────────────┘   │
│                                 │
│  ☐ 비활성화                     │  (루틴 수정 시에만)
│                                 │
│  세부 항목                       │  제목 (bold)
│                                 │
│  ↳ 스트레칭 10분          ✕    │  기존 세부 항목
│  ↳ 러닝 30분             ✕    │  "✕" = 삭제 버튼
│                                 │
│  ┌──────────────────┐ ⊕       │
│  │ 세부 항목 추가      │         │  TextField + 추가 버튼
│  └──────────────────┘          │
│                                 │
│  ┌─────────────────────────┐   │
│  │         저장              │   │  ElevatedButton
│  └─────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
```

### 세부 항목 섹션 UI 요소
- **섹션 제목**: "세부 항목" (titleSmall, bold)
- **기존 항목 목록**: ListTile, dense
  - leading: ↳ 아이콘 (subdirectory_arrow_right, 18px, grey[500])
  - title: 항목 텍스트
  - trailing: ✕ 닫기 버튼 (IconButton, close, 18px)
- **추가 입력**: Row
  - TextField (hintText: "세부 항목 추가", maxLength 30, UnderlineInputBorder)
  - IconButton (add_circle_outline)
  - Enter 키로도 추가 가능

## 홈 화면 - 세부 항목 인라인 표시

```
┌─────────────────────────────────┐
│ ○ 아침 운동              2  🔴 │  체크 + 제목 + 세부항목수 + 카테고리
│    ↳ 스트레칭 10분              │  세부 항목 (indent, 13px)
│    ↳ 러닝 30분                  │
├─────────────────────────────────┤
│ ✓ 영어 공부              1  🔵 │  완료 상태
│    ↳ 단어 50개 외우기           │  취소선 + grey[400]
└─────────────────────────────────┘
```

### 홈 화면 타일 UI 요소
- **세부 항목 수 뱃지**: subtasks.isNotEmpty일 때 trailing에 숫자 표시 (12px, grey[500])
- **세부 항목 인라인**: Padding(left: 56)
  - Row: ↳ 아이콘 (14px, grey[400]) + SizedBox(6) + Text (13px)
  - 완료 시: grey[400] + lineThrough
  - 미완료 시: grey[600]

## 인터랙션 상세

### 탭 동작 구분
| 탭 영역 | 동작 |
|---|---|
| 체크 아이콘 (leading GestureDetector) | 완료/미완료 토글 |
| 나머지 영역 (ListTile onTap) | 수정 바텀시트 열기 |

### 수정 후 동작
- 루틴 수정: title, categoryId, isActive, subtasks 업데이트
- 할 일 수정: title, categoryId, subtasks 업데이트
- 루틴 비활성화 시: DailyRecord에서 해당 루틴 completion 제거

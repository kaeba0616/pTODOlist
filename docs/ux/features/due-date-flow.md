# F12: 할 일 마감일 - UX Flow

## 개요
할 일에 미래 날짜를 지정하고, 미완료 지난 할 일을 overdue로 표시한다.

---

## 플로우 1: 미래 날짜로 할 일 추가
```
할 일 추가 바텀시트
→ 제목, 카테고리, 세부 항목 (기존)
→ 날짜 선택 (기본=오늘, 미래 선택 가능)
    → 날짜 칩 탭 → DatePicker
→ 저장
```

## 플로우 2: 홈 화면에서 overdue 표시
```
홈 화면 → "추가 할 일" 섹션
→ 미완료 + 지난 날짜 할 일이 상단에 "D+N" 뱃지와 함께 표시
→ 오늘 할 일이 그 아래 표시
→ 완료된 할 일은 최하단
```

---

## 규칙
- 날짜 기본값: 오늘
- 과거 날짜 선택 불가 (오늘 이후만)
- overdue: targetDate < 오늘 && isCompleted == false
- overdue 정렬: 오래된 것부터 (targetDate 오름차순)
- overdue 할 일의 "D+N" 뱃지: 오늘 - targetDate 일수

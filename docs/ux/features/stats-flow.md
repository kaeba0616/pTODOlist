# F6: 달성률 통계 - UX Flow

## 진입 경로
바텀 탭 "통계" 탭

---

## 화면 구성
- 상단: 기간 선택 탭 (일별 / 주별 / 월별)
- 중앙: 막대 차트 (달성률 %)
- 하단: 카테고리별 달성률 분석

## 플로우 1: 일별 통계 (기본)
```
통계 탭 → 최근 7일 막대 차트 표시
  - X축: 날짜 (월, 화, ...)
  - Y축: 달성률 0~100%
  - 오늘은 강조 색상
```

## 플로우 2: 주별 통계
```
"주별" 탭 선택 → 최근 4주 막대 차트
  - X축: n주 전
  - Y축: 주간 평균 달성률
```

## 플로우 3: 월별 통계
```
"월별" 탭 선택 → 최근 6개월 막대 차트
  - X축: 월 (1월, 2월, ...)
  - Y축: 월간 평균 달성률
```

## 달성률 계산
```
일별 달성률 = (완료 루틴 + 완료 할일) / (전체 루틴 + 전체 할일) × 100
주별 = 해당 주 일별 달성률의 평균
월별 = 해당 월 일별 달성률의 평균
```
- DailyRecord가 없는 날은 0%로 처리
- 루틴+할일이 0개인 날은 제외

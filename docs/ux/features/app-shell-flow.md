# F0: 앱 셸 & 네비게이션 - UX Flow

## 앱 진입 플로우

```
앱 실행
  │
  ├── Hive 초기화
  ├── AppSettings 로드
  ├── MidnightResetService 시작
  │
  └── 홈 화면 (오늘 탭) 표시
```

- 스플래시/로딩 없음 (Hive 초기화는 매우 빠름)
- 로그인/온보딩 없음 (제로 프릭션)
- 항상 홈 화면(오늘 탭)으로 진입

---

## 탭 네비게이션

```
┌─────────────────────────────────┐
│                                 │
│        [현재 탭 화면 영역]        │
│                                 │
│                                 │
│                                 │
│                            [+]  │  ← FAB (홈 탭에서만 표시)
├─────────────────────────────────┤
│   🏠 오늘    📊 통계    ⚙ 설정   │  ← 바텀 네비게이션
└─────────────────────────────────┘
```

### 탭 구성
| 순서 | 탭 | 아이콘 | 라벨 | 설명 |
|---|---|---|---|---|
| 1 | 오늘 | Icons.home_rounded | 오늘 | 오늘의 루틴 + 할 일 (기본 탭) |
| 2 | 통계 | Icons.bar_chart_rounded | 통계 | 달성률 차트 (Phase 2) |
| 3 | 설정 | Icons.settings_rounded | 설정 | 카테고리, 알림, 데이터 관리 |

### 탭 전환 동작
- 탭 간 전환 시 상태 유지 (각 탭은 독립적 Navigator)
- 활성 탭: Primary 색상
- 비활성 탭: Gray 400

---

## 라우트 구조

```
/                    → 홈 화면 (ShellRoute 내부)
/stats               → 통계 화면 (ShellRoute 내부)
/settings            → 설정 화면 (ShellRoute 내부)
/settings/categories → 카테고리 관리 (push)
/add-routine         → 루틴 추가 (모달 바텀시트)
/add-task            → 할 일 추가 (모달 바텀시트)
```

- ShellRoute: 바텀 네비게이션을 감싸는 공통 쉘
- 모달 화면: 바텀시트로 표현 (별도 라우트 또는 showModalBottomSheet)

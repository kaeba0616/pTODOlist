# F11: 연속 달성 추적 (Streak) - UX Flow

## 개요
루틴의 연속 완료 일수를 추적하여 홈 화면에 불꽃 뱃지로 표시한다.

## 표시 규칙
- 연속 달성 2일 이상: 불꽃 아이콘 + 일수 표시
- 연속이 끊기면 0으로 리셋
- 루틴이 비활성인 날은 연속에 영향 없음 (요일별 루틴 반영)
- 루틴이 아직 존재하지 않던 날도 연속에 영향 없음

## 계산 방식
- DailyRecord에서 해당 routineId의 완료 여부를 역순으로 조회
- 활성 요일이 아닌 날은 건너뜀 (streak 유지)
- 미완료일 발견 시 중단

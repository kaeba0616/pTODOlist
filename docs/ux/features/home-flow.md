# F4: 오늘 화면 - UX Flow

## 화면 구성
홈 화면은 앱의 메인 화면으로, 오늘의 루틴과 할 일을 한 눈에 보여준다.

---

## 플로우 1: 루틴 체크
```
홈 화면 → 루틴 체크박스 탭 → 완료 애니메이션 → 진행률 업데이트
```

## 플로우 2: 할 일 체크
```
홈 화면 → 할 일 체크박스 탭 → 완료 (취소선+투명도) → 하단 이동 → 진행률 업데이트
```

## 플로우 3: 항목 추가
```
홈 화면 → FAB(+) 탭 → 바텀시트
  ├── "루틴 추가" → RoutineFormView
  └── "할 일 추가" → TaskFormView
```

## 플로우 4: 항목 삭제
```
홈 화면 → 항목 왼쪽 스와이프 → 삭제
```

## 플로우 5: 루틴 수정
```
홈 화면 → 루틴 항목 길게 누르기 → RoutineFormView (수정 모드)
```

---

## 진행률 계산
```
전체 = 활성 루틴 수 + 오늘 할 일 수
완료 = 완료된 루틴 수 + 완료된 할 일 수
진행률 = 완료 / 전체 * 100
```
- 항목이 0개면 진행률 표시 안 함

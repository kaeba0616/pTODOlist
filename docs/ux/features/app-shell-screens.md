# F0: 앱 셸 & 네비게이션 - Screens

## 공통 쉘 (ScaffoldWithNavBar)

```
┌─────────────────────────────────┐
│  [StatusBar - 시스템]             │
├─────────────────────────────────┤
│                                 │
│                                 │
│        [Child Screen]           │
│        (탭별 화면이 여기에)        │
│                                 │
│                                 │
│                                 │
├─────────────────────────────────┤
│  🏠 오늘     📊 통계     ⚙ 설정  │
│  (active)   (inactive) (inactive)│
└─────────────────────────────────┘
```

### UI 요소
- **바텀 네비게이션 바**
  - 높이: 80dp (Material 3 기본)
  - 배경: Surface 색상
  - 상단 경계선: 없음 (elevation으로 구분)
  - 아이콘 크기: 24dp
  - 라벨: labelSmall (12sp)
  - 활성 색상: Primary (#4F46E5)
  - 비활성 색상: Gray 400 (#9CA3AF)

- 다크 모드: 시스템 설정 따름 (ThemeMode.system)

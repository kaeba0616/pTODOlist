# F3: 추가 할 일 - UX Flow

## 진입 경로
- 홈 화면 FAB(+) → "할 일 추가" 선택

---

## 플로우 1: 할 일 추가
```
홈 화면 → FAB(+) → 바텀시트 "할 일 추가" 선택
→ 할 일 추가 바텀시트
  → 제목 입력 (필수)
  → 카테고리 선택 (기본: 기타)
→ 홈 화면 "추가 할 일" 섹션에 추가됨
```

## 플로우 2: 할 일 완료 토글
```
홈 화면 → 할 일 체크박스 탭
→ 완료 시: 취소선 + 투명도 감소 + 섹션 하단 이동
→ 미완료로 되돌리기: 다시 탭하면 원래 상태
```

## 플로우 3: 할 일 삭제
```
홈 화면 → 할 일 항목 왼쪽 스와이프 → 삭제
```

---

## 루틴과의 차이점
| 항목 | 루틴 | 추가 할 일 |
|---|---|---|
| 반복 | 매일 자동 | 일회성 |
| 자정 초기화 | O | X |
| 완료 상태 저장 | DailyRecord | 자체 isCompleted |
| 날짜 | 매일 동일 | targetDate 지정 |

# F2: 루틴 관리 - Screens

## 루틴 추가/수정 바텀시트

```
┌─────────────────────────────────┐
│  ─── (드래그 핸들)               │
│                                 │
│  루틴 추가                       │  제목 (수정 시 "루틴 수정")
│                                 │
│  제목                           │
│  ┌─────────────────────────┐   │
│  │ 아침 운동                 │   │  TextField
│  └─────────────────────────┘   │
│                                 │
│  카테고리                        │
│  ┌─────────────────────────┐   │
│  │ 🔴 운동              ▼  │   │  DropdownButton
│  └─────────────────────────┘   │
│                                 │
│  ☐ 비활성화                     │  (수정 시에만 표시)
│                                 │
│  ┌─────────────────────────┐   │
│  │         저장              │   │  ElevatedButton
│  └─────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
```

### UI 요소
- **제목 입력**: TextField, maxLength 30, 빈 값 시 저장 비활성
- **카테고리 선택**: DropdownButton, 기존 카테고리 목록에서 선택
- **비활성화 체크박스**: 수정 시에만 노출, 비활성 시 홈 화면에서 숨김
- **저장 버튼**: 제목 비어있으면 비활성

## 홈 화면 내 루틴 타일 (RoutineTile)

```
┌─────────────────────────────────┐
│  ○  아침 운동            🔴     │  미완료 상태
│  ◉  영어 공부            🔵     │  완료 상태 (취소선)
└─────────────────────────────────┘
```

- 왼쪽: 체크박스 (원형)
- 중앙: 루틴 제목 (완료 시 취소선 + 투명도)
- 우측: 카테고리 색상 도트
- 길게 누르기: 수정 바텀시트
- 왼쪽 스와이프: 삭제

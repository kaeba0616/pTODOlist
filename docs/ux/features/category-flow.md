# F1: 카테고리 관리 - UX Flow

## 진입 경로
```
설정 탭 → "카테고리 관리" 메뉴 → 카테고리 목록 화면
```

---

## 플로우 1: 카테고리 목록 보기
```
설정 화면 → "카테고리 관리" 탭 → 카테고리 목록 표시
  - 각 항목: 색상 도트 + 이름 + 연결된 루틴/할일 수
  - 하단: "카테고리 추가" 버튼
```

## 플로우 2: 카테고리 추가
```
카테고리 목록 → "+" 버튼 → 추가 바텀시트
  → 이름 입력 (필수)
  → 색상 선택 (프리셋 팔레트)
  → "저장" 버튼
  → 목록에 새 항목 추가됨
```

## 플로우 3: 카테고리 수정
```
카테고리 목록 → 항목 탭 → 수정 바텀시트
  → 이름 수정
  → 색상 변경
  → "저장" 버튼
  → 목록 업데이트
```

## 플로우 4: 카테고리 삭제
```
카테고리 목록 → 항목 왼쪽 스와이프 → 삭제 버튼
  → "기타" 카테고리인 경우: 삭제 불가 스낵바
  → 그 외: 확인 다이얼로그 "이 카테고리의 루틴/할 일은 '기타'로 이동됩니다"
    → 확인 → 삭제 + 연관 항목 재할당
    → 취소 → 아무 동작 없음
```

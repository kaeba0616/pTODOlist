# F2: 루틴 관리 - UX Flow

## 진입 경로
- 홈 화면 FAB(+) → "루틴 추가" 선택
- 홈 화면 루틴 항목 탭 → 수정
- 홈 화면 루틴 항목 스와이프 → 삭제

---

## 플로우 1: 루틴 추가
```
홈 화면 → FAB(+) → 바텀시트 "루틴 추가" 선택
→ 루틴 추가 바텀시트
  → 제목 입력 (필수)
  → 카테고리 선택 (드롭다운, 기본값: 기타)
  → "저장" 버튼
→ 홈 화면 루틴 목록에 새 항목 추가됨
```

## 플로우 2: 루틴 수정
```
홈 화면 → 루틴 항목 길게 누르기 → 수정 바텀시트
→ 제목 수정
→ 카테고리 변경
→ "저장" 버튼
→ 목록 업데이트
```

## 플로우 3: 루틴 삭제
```
홈 화면 → 루틴 항목 왼쪽 스와이프 → 삭제 버튼
→ 확인 다이얼로그 "이 루틴을 삭제하시겠습니까?"
  → 확인 → 삭제
  → 취소 → 유지
```

## 플로우 4: 루틴 비활성화
```
홈 화면 → 루틴 항목 길게 누르기 → 수정 바텀시트
→ "비활성화" 토글
→ 비활성화된 루틴은 홈 화면에서 숨겨짐
```

---

## 루틴 속성
| 필드 | 필수 | 설명 |
|---|---|---|
| title | O | 루틴 제목 (최대 30자) |
| categoryId | O | 카테고리 (기본: 기타) |
| isActive | - | 활성 여부 (기본: true) |
| order | - | 정렬 순서 (추가 순서) |

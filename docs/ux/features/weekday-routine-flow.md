# F10: 요일별 루틴 - UX Flow

## 개요
루틴에 활성 요일을 설정하여, 해당 요일에만 홈 화면에 표시된다.
예: "헬스"는 월/수/금만, "영어"는 매일.

---

## 플로우 1: 요일 지정하여 루틴 추가
```
루틴 추가 바텀시트
→ 제목, 카테고리, 아이콘, 우선순위 (기존)
→ 활성 요일 선택
    → 7개 토글 칩 (월~일), 기본=전체 선택
    → 탭하여 ON/OFF 토글
    → 최소 1개 이상 선택 필수
→ 저장
```

## 플로우 2: 요일 변경
```
홈 화면 → 루틴 탭 → 수정 바텀시트
→ 기존 활성 요일이 미리 선택됨
→ 변경 후 저장
```

---

## 요일 규칙
- `activeDays`: List<int> (1=월 ~ 7=일, Dart DateTime.weekday 기준)
- 빈 리스트 `[]` = 매일 (하위 호환성, 기존 데이터)
- 전체 선택 시에도 `[]`로 저장 (= 매일)
- 최소 1개 요일은 선택되어야 함

## 홈 화면 필터링
- 오늘 요일에 해당하는 루틴만 표시
- `activeDays`가 비어있으면 항상 표시
- DailyRecord 생성 시에도 오늘 요일 기준으로 루틴 포함

## 통계 영향
- 각 날의 DailyRecord에는 해당 날 활성이었던 루틴만 기록
- 달성률 = 완료 수 / 해당 날 활성 루틴 수

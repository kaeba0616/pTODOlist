# pTODOlist - Vision One-Pager

## 한 줄 미션
> 매일의 루틴을 체크하고, 완료하지 못한 일을 놓치지 않도록 도와주는 가장 심플한 습관 관리 앱

---

## 문제 정의
- 매일 해야 하는 일(루틴)을 머릿속으로만 관리하면 빠뜨리기 쉬움
- 기존 TODO 앱은 일회성 할 일에 초점 → 반복 루틴 관리에 부적합
- 습관 트래커 앱은 기능이 많고 복잡하여 오히려 사용을 포기하게 됨
- 하루가 끝나기 전 남은 일을 알려주는 기능이 부족

## 솔루션
- **매일 루틴**: 매일 반복되는 할 일을 등록하면 자정에 자동 초기화
- **추가 할 일**: 오늘만 해야 하는 일회성 태스크 관리
- **카테고리 분류**: 운동, 공부, 업무 등으로 구분
- **달성률 통계**: 일별/주별/월별 완료율을 차트로 확인
- **11시 스마트 알림**: 자정 1시간 전, 미완료 항목만 알려줌 (전부 완료면 무음)

## 타겟 유저
- 매일의 루틴을 꾸준히 실천하고 싶은 개인
- 복잡한 앱보다 심플한 체크리스트를 원하는 사람
- 자기 관리/습관 형성에 관심 있는 20~40대

## 핵심 가치
1. **심플함**: 열면 바로 오늘의 할 일이 보임
2. **자동화**: 자정 자동 초기화 → 매일 새로 시작
3. **프라이버시**: 로컬 전용 → 서버 없음, 계정 없음, 데이터는 내 폰에만
4. **동기부여**: 달성률 통계로 꾸준함을 시각화

## 차별화 포인트
| 기존 앱 | pTODOlist |
|---|---|
| 서버 필수, 가입 필요 | 로컬 전용, 가입 불필요 |
| 기능 과다 (팀, 프로젝트, 반복설정 등) | 루틴 + 할 일 2가지에 집중 |
| 수동 초기화 | 자정 자동 초기화 |
| 알림 = 고정 시간 리마인더 | 스마트 알림 = 남은 일이 있을 때만 |

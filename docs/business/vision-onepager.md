# pTODOlist - Vision One-Pager

## 한 줄 미션
> 혼자서도, 친구와 함께도 — 매일의 습관을 만드는 올인원 트래커

---

## 문제 정의
- 매일 해야 하는 일(루틴)을 머릿속으로만 관리하면 빠뜨리기 쉬움
- 기존 TODO 앱은 일회성 할 일에 초점 → 반복 루틴 관리에 부적합
- 습관 트래커 앱은 기능이 많고 복잡하여 오히려 사용을 포기하게 됨
- 하루가 끝나기 전 남은 일을 알려주는 기능이 부족
- 혼자만 보는 트래커는 동기부여가 약하지만, 소셜 앱은 너무 무거움

## 솔루션
- **매일 루틴 + 요일별 활성화**: 매일 / 평일 / 주말 / 특정 요일에 자동 활성
- **추가 할 일 + 마감일**: 일회성 태스크와 overdue 표시
- **카테고리 + subtask**: 운동, 공부, 업무 등 그룹화 + 세부 항목
- **자정 자동 초기화**: DailyRecord 패턴으로 매일 새로 시작
- **시각화**: 일/주/월 통계 + 캘린더 히트맵 + Android 홈 위젯
- **스마트 알림**: 11시 미완료 항목만 (전부 완료 시 무음)
- **선택적 클라우드 + 친구 공유**: Google 로그인 시 멀티 디바이스 + 친구의 루틴/진행률 확인

## 타겟 유저
- 매일의 루틴을 꾸준히 실천하고 싶은 개인
- 복잡한 앱보다 심플한 체크리스트를 원하는 사람
- 자기 관리/습관 형성에 관심 있는 20~40대
- 친구와 함께 동기부여 받고 싶지만 게임화는 부담스러운 사람

## 핵심 가치
1. **심플함**: 열면 바로 오늘의 할 일이 보임
2. **자동화**: 자정 리셋 + 11시 스마트 알림 — 매일 새로 시작
3. **시각화 + 모티베이션**: 통계 + 캘린더 히트맵 + 위젯 + 친구의 진행률
4. **유연한 시작**: 로그인 없이도 100% 동작 — 원하면 Google 로그인으로 멀티 디바이스 + 친구 공유 (프라이버시 옵션 유지)

## 차별화 포인트

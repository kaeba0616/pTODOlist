## 2. 포지셔닝

### 포지셔닝 맵
```
         복잡 (기능 많음)
           │
  Todoist  │  TickTick
           │
  ─────────┼──────────── 로컬 ←→ 서버
           │
  Habitica │
           │       pTODOlist ★ (옵션형)
  Streaks  │
         심플 (기능 적음)
```

- pTODOlist 는 더 이상 Streaks 옆 끝단이 아님
- 심플 사이드에 있지만 친구/클라우드 기능으로 한 칸 위
- 로컬↔서버 축은 가운데 (둘 다 가능)

### 핵심 포지셔닝
> **"심플함을 잃지 않은 올인원 습관 트래커"**
>
> 가입 없이 시작해도 루틴/할일/통계/위젯/캘린더가 다 동작.
> 친구가 생기면 로그인하고 코드 교환 → 함께 동기부여.
> 자정에 자동 초기화, 자기 전에 남은 일만 알려줌.

### 차별화 3가지
1. **심플 + 올인원**: 루틴/할일/통계/위젯/캘린더/친구 — 다 있지만 복잡하지 않음
2. **스마트 무음**: 다 했으면 알림 안 울림 → 알림 피로 없음
3. **유연한 시작**: 가입 없이도 100% 동작 + Google 로그인으로 멀티 디바이스/친구 공유 (프라이버시 옵션 유지)

## 3. 비즈니스 모델

### 현재 (v1.0~Phase 6)
- **무료 배포**: 핵심 기능 (루틴/할일/자정 리셋/알림/통계/위젯/캘린더/클라우드/친구) 모두 무료
- **비용 구조**: Firebase 무료 한도 내 운영 → 활성 사용자 100명까지 $0

### 향후 수익화 가능성

| 모델 | 내용 | 예상 가격 |
|---|---|---|
| 프리미엄 테마 | 커스텀 색상/아이콘 팩 | ₩3,900 일회성 |
| 고급 통계 | 연간 리포트, 카테고리 심층 분석 | ₩2,900/년 |
| 친구 수 무제한 | 무료는 N명 제한 (수치는 추후 결정) | ₩1,900/월 |
| AI 코칭/추천 | 루틴 추천 + 패턴 분석 | ₩4,900/월 |

### 수익화 원칙
- 핵심 기능은 **영구 무료** (이미 무료로 제공 중인 항목 유료화 X)
- 프리미엄은 편의/꾸미기/확장 기능에만 적용
- 광고 없음 (사용자 경험 최우선)
- Firebase 비용이 매월 ~$100 도달하기 전까지 유료화는 보류

### 수익화 트리거 시점
- 활성 사용자 ~10,000명 도달 (Firebase 월 비용 ~$100)
- 그 전에는 무료 + 사용자 확보에 집중

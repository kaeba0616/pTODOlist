## 2. 포지셔닝

### 포지셔닝 맵
```
         복잡 (기능 많음)
           │
  Todoist  │  TickTick
           │
  ─────────┼──────────── 서버 ←→ 로컬
           │
  Habitica │  pTODOlist ★
           │  Streaks
         심플 (기능 적음)
```

### 핵심 포지셔닝
> **"가장 심플한 로컬 전용 루틴 관리 앱"**
>
> 가입 없이, 서버 없이, 열면 바로 오늘의 루틴을 체크.
> 자정에 자동 초기화되고, 자기 전에 남은 일만 알려줌.

### 차별화 3가지
1. **제로 프릭션**: 가입/로그인/서버 없음 → 설치 즉시 사용
2. **스마트 무음**: 다 했으면 알림 안 울림 → 알림 피로 없음
3. **프라이버시 퍼스트**: 데이터가 폰 밖으로 나가지 않음

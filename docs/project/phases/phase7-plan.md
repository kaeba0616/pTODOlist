**목표**: Phase 1~6 의 코드 현실을 비즈니스 문서에 반영하고,
"올인원 습관 트래커" 톤으로 비전 / 포지셔닝 / BM 을 일관되게 재정렬

**상태**: `[status: done]`

**유형**: 순수 전략 Phase (코드 변경 없음, 문서만)

---

## 배경

Phase 1~6 까지 진행하면서 사업계획서 / vision / positioning 문서가 코드 현실과
어긋나게 됨. 특히 `business-plan.md` 의 **WON'T (현재 범위 밖)** 섹션 4개 항목이
모두 이미 구현됨:

| 기존 WON'T | 실제 구현 |
|---|---|
| 서버/클라우드 동기화 | F15 (Phase 6) |
| 소셜 기능 (친구, 공유) | F16 (Phase 6) |
| 요일별 다른 루틴 | F10 (Phase 4) |
| 위젯 | F14 (Phase 5) |

또한 `vision-onepager.md` 의 핵심 가치 "프라이버시 (로컬 전용, 데이터는 내 폰에만)" 가
F15 이후로 더 이상 정확하지 않음.

이 Phase 는 코드를 건드리지 않고, 비즈니스/제품/포지셔닝 문서를
**"심플함을 잃지 않은 올인원 습관 트래커"** 톤으로 일관되게 재작성한다.

## 핵심 결정 (브레인스토밍 합의)

1. **비전 톤**: "올인원 습관 트래커" 로 격상. 포지셔닝 카테고리가 Streaks 끝단에서
   심플 사이드 중간 (Streaks ↔ Todoist 사이) 으로 이동
2. **프라이버시**: 핵심 가치에서 빼되 **"유연한 시작"** 의 한 축으로 살림 (로그인 없이도 100% 동작)
3. **한 줄 미션**: "혼자서도, 친구와 함께도 — 매일의 습관을 만드는 올인원 트래커"
4. **한 줄 포지셔닝**: "심플함을 잃지 않은 올인원 습관 트래커"
5. **새 핵심 가치 4개**: 심플함 / 자동화 / 시각화+모티베이션 / 유연한 시작
6. **새 차별화 3개**: 심플 + 올인원 / 스마트 무음 / 유연한 시작
7. **새 WON'T**: 팀 협업 / 게이미피케이션 / 캘린더 일정 / 광고

## Workstream 1: vision-onepager.md

### 한 줄 미션
> "혼자서도, 친구와 함께도 — 매일의 습관을 만드는 올인원 트래커"

### 솔루션 (재작성)
- **매일 루틴 + 요일별 활성화**: 매일 / 평일 / 주말 / 특정 요일에 자동 활성
- **추가 할 일 + 마감일**: 일회성 태스크와 overdue 표시
- **카테고리 + subtask**: 그룹화 + 세부 항목
- **자정 자동 초기화**: DailyRecord 패턴
- **시각화**: 일/주/월 통계 + 캘린더 히트맵 + 홈 위젯 (Android)
- **스마트 알림**: 11시 미완료 항목만 (전부 완료 시 무음)
- **선택적 클라우드 + 친구 공유**: Google 로그인 시 멀티 디바이스 + 친구의 루틴/진행률 확인

### 핵심 가치 4개 (재작성)
1. **심플함** — 열면 바로 오늘의 할 일
2. **자동화** — 자정 리셋 + 11시 스마트 알림
3. **시각화 + 모티베이션** — 통계 + 캘린더 + 위젯 + 친구의 진행률
4. **유연한 시작** — 로그인 없이도 100% 동작, 원하면 Google 로그인 + 친구 공유 (프라이버시 옵션은 살아있음)

### 차별화 포인트 표 (기존 5컬럼 표 업데이트)
- pTODOlist 컬럼의 "서버 X, 가입 X" → "서버/가입 선택"
- 새 행: "친구 공유: pTODOlist O, 다른 앱 X/△"

## Workstream 2: business-plan.md

### 한 줄 소개
> "혼자서도 친구와 함께도, 매일의 습관을 만드는 올인원 모바일 앱"

### 비전 (재작성)
> 누구나 쉽게 매일의 루틴을 관리하고, 통계·캘린더·위젯·친구의 응원으로
> 꾸준한 습관을 만들어갈 수 있는 가장 가벼운 올인원 도구

### 페인 포인트 (5번 프라이버시 → 톤 조정)
- 5번 "프라이버시 우려" → "**서버에 일정 올리는 부담**: 서버 필수 앱은
  계정 만들고 데이터 올려야 함 → 가입 없이 시작하고, 원할 때만 동기화 켜고 싶음"

### MoSCoW 재정렬

```
DONE (v1.0 ~ Phase 6 완료):
- Phase 1: 루틴/할일/카테고리/자정 리셋/달성률
- Phase 2: 통계 + 11시 스마트 알림
- Phase 3: 설정/데이터 관리/다크모드
- Phase 4: subtask/요일별 루틴/streak/마감일
- Phase 5: 캘린더 히트맵/Android 홈 위젯
- Phase 6: Google 로그인 + 클라우드 동기화 + 친구 공유

MUST (다음 마일스톤):
- iOS 홈 위젯
- iOS Firebase 통합

SHOULD:
- 친구 상호작용 강화 (응원/주간 진행률/친구 streak)
- 알림 시간 다양화

COULD:
- 유료화 (프리미엄 테마, 고급 통계)
- AI 코칭/추천

WON'T (의도적 비범위):
- 팀 협업/프로젝트 관리 (Todoist 영역)
- 게이미피케이션 (Habitica 영역)
- 캘린더 일정/시간 블록 (Sunsama 영역)
- 광고
```

### 페르소나 (1명 추가)

**박지영 (25세, 대학생)** — 신규
> 친구들과 함께 운동 루틴을 시작했다. 서로 얼마나 꾸준히 하는지 보면 의지가 생긴다.
> 가입 없이 시작하다가 친구가 생기면 그때 로그인하고 코드 교환하면 좋겠다.

(김민수 28세 페르소나는 유지)

### 기술 전략 (Backend 항목 업데이트)
- "Backend: 없음 (로컬 전용)" → "**Backend: 선택적 Firebase (Auth + Firestore)**
  — 로그인 안 하면 로컬 100% 동작, 로그인 시 멀티 디바이스 + 친구 공유"

### 수익 모델 (향후 가능성 표 업데이트)
- ❌ 위젯 ₩1,900 일회성 — 제거 (이미 무료)
- ❌ 클라우드 백업 ₩4,900/년 — 제거 (이미 무료)
- ✅ 프리미엄 테마 ₩3,900 일회성 — 유지
- ✅ 고급 통계 ₩2,900/년 — 유지
- ➕ 친구 수 무제한 (무료는 N명 제한 — 정확한 수치는 추후 결정)
- ➕ AI 코칭/추천 ₩4,900/월

### 로드맵 표 업데이트
- Phase 4~6 추가 (Phase 1~3 만 있던 표를 1~6 으로 확장)
